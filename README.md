Equalizer - adjust all tones to own preferences, save custom presets

## Features
- 5 bands configuration
- Saving custom presets
- Additional volume adjustment
- System notification

## Built With
- [Android SDK](https://developer.android.com/reference/packages)
- [Kotlin](https://kotlinlang.org/)
- [Coroutines](https://github.com/Kotlin/kotlinx.coroutines) 
- [Hilt](https://developer.android.com/training/dependency-injection/hilt-android)
- [Jetpack Compose](https://developer.android.com/compose)
- [Material 3 Expressive](https://m3.material.io/blog/building-with-m3-expressive)
- [Data Store](https://developer.android.com/topic/libraries/architecture/datastore)
- [Dynamic colors](https://developer.android.com/develop/ui/views/theming/dynamic-colors)
- [Adaptive icons](https://developer.android.com/develop/ui/views/launch/icon_design_adaptive)
- [Mockito](https://site.mockito.org/)
